# 모임 이벤트 관리 웹 MVP PRD

## 핵심 정보

**목적**: 카카오톡 채팅에 묻히는 공지/참여 확인/정산을 한 곳에서 관리해 소규모 모임 주최자의 운영 부담을 줄인다.
**사용자**: 수영·헬스 등 소규모 동호회나 지인 모임을 이끄는 주최자, 링크로 간단히 참여 응답하는 참여자.

---

## 사용자 여정

### 주최자 여정

```
1. 홈 페이지
   ↓ "시작하기" 버튼 클릭

2. 로그인 페이지
   ↓ Google 로그인 완료 → 자동 리디렉션

3. 이벤트 목록 페이지
   ↓ "이벤트 만들기" 버튼 클릭

4. 이벤트 생성 페이지
   ↓ 폼 작성 후 "생성하기" 버튼 클릭

5. 이벤트 상세 페이지 (공개)
   ↓ "공유 링크 복사" → 카카오톡/문자 공유

6. 이벤트 관리 페이지
   ↓ 참여자 명단 확인 / 공지 등록 / 이벤트 수정
```

### 참여자 여정

```
1. 공유 링크 클릭 → 이벤트 상세 페이지 (비회원 접근)
   ↓ 공지 및 이벤트 정보 확인

2. 이벤트 상세 페이지 — 참여 응답 폼
   ↓ 이름 입력 + 참여/불참/미정 선택 + (선택) 한마디 메모 입력

   [정원 여유] → 참여 명단 등록 → 명단 즉시 반영
   [정원 초과] → 대기 명단 자동 등록 → 안내 메시지 표시

3. 완료 → 참여자 명단에서 본인 확인
```

## 기능 명세

### 1. MVP 핵심 기능

| **F003** | 이벤트 수정/삭제        | 주최자가 이벤트 정보를 수정하거나 이벤트를 삭제                    | 날짜·장소 변경 등 실운영에 반드시 필요                   | 이벤트 관리 페이지                     |
| **F004** | 공유 링크 생성          | 이벤트별 고유 토큰 기반 공개 URL 생성 및 클립보드 복사             | 비회원 참여자가 링크 하나로 접근 가능해야 핵심 가치 실현 | 이벤트 상세 페이지, 이벤트 관리 페이지 |
| **F005** | 공지사항 등록/수정/삭제 | 주최자가 이벤트별 공지를 등록하고 최신 공지를 상단 고정            | 채팅에 묻히던 공지를 별도 관리해 참여자가 항상 확인 가능 | 이벤트 상세 페이지, 이벤트 관리 페이지 |
| **F006** | 참여 응답 (비회원)      | 이름 입력 후 참여/불참/미정 선택, 한마디 메모 선택 입력            | 앱 설치·회원가입 없이 30초 내 응답이 참여자 핵심 니즈    | 이벤트 상세 페이지                     |
| **F007** | 참여자 명단 조회        | 참여/불참/미정/대기 상태별 명단 실시간 확인                        | 주최자가 인원을 파악하는 핵심 기능                       | 이벤트 상세 페이지, 이벤트 관리 페이지 |
| **F008** | 대기 명단 자동 관리     | 최대 인원 초과 시 대기 등록, 취소 발생 시 자동 순번 승격           | 정원 관리 없으면 인원 초과 혼선 발생                     | 이벤트 상세 페이지, 이벤트 관리 페이지 |
| **F009** | 참여 응답 수정 (주최자) | 주최자가 참여자의 응답 상태를 직접 수정                            | 단순 UX 유지 및 분쟁 방지 정책 반영                      | 이벤트 관리 페이지                     |

```

헤더 메뉴 (비로그인)
├── 로고/홈 → 홈 페이지

헤더 메뉴 (로그인 후 — 주최자)
├── 로고/홈 → 홈 페이지

주최자 메뉴 (이벤트 목록 페이지)
├── 이벤트 목록 → 이벤트 목록 페이지 (F002)
└── 이벤트 만들기 → 이벤트 생성 페이지 (F001)

이벤트 상세 페이지 내 액션 (공개 — 비회원/주최자 공통)
├── 공지사항 탭 (F005)
├── 참여자 명단 탭 (F007, F008)

└── 공유 링크 복사 버튼 (F004)

이벤트 관리 페이지 내 액션 (주최자 전용)
├── 참여자 관리 탭 (F007, F008, F009)
├── 공지 등록/수정/삭제 탭 (F005)
├── 이벤트 수정 탭 (F003)
├── 공유 링크 복사 (F004)
└── 이벤트 삭제 (F003)
```

## 페이지별 상세 기능

### 홈 페이지

> **구현 기능:** `F010` | **접근:** 비로그인/로그인 모두 가능

| 항목            | 내용                                                                                                                                         |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 서비스 랜딩 페이지. 가치 제안을 한눈에 전달하고 주최자를 로그인으로 유도                                                                     |
| **진입 경로**   | 직접 URL 접속, 로고 클릭, 로그아웃 후 자동 리디렉션                                                                                          |
| **사용자 행동** | 서비스 소개 확인 후 "시작하기" 버튼 클릭해 로그인 페이지로 이동                                                                              |
| **주요 기능**   | - 서비스 핵심 기능 소개 (공지/참여 관리/공유 링크)<br>- 카카오톡 대비 문제 해결 가치 제안 문구<br>- **"시작하기"** 버튼 → 로그인 페이지 이동 |
| **다음 이동**   | "시작하기" 클릭 → 로그인 페이지, 이미 로그인 상태 → 이벤트 목록 페이지 자동 리디렉션                                                         |

---

### 로그인 페이지

> **구현 기능:** `F010` | **접근:** 비로그인 사용자만

| 항목            | 내용                                                                                                                                                                 |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 주최자 인증 전용 페이지. Google OAuth 로그인만 지원                                                                                                                  |
| **진입 경로**   | 홈 페이지 "시작하기" 버튼, 헤더 "로그인" 메뉴, 보호된 페이지 접근 시 자동 리디렉션                                                                                   |
| **사용자 행동** | "Google로 계속하기" 버튼 클릭 → Google OAuth 인증 완료                                                                                                               |
| **주요 기능**   | - **"Google로 계속하기"** 버튼 (Google OAuth 연동)<br>- 로그인 성공 시 이벤트 목록 페이지로 자동 리디렉션<br>- 이미 로그인 상태면 이벤트 목록 페이지로 즉시 리디렉션 |
| **다음 이동**   | 로그인 성공 → 이벤트 목록 페이지                                                                                                                                     |

---

### 이벤트 목록 페이지

> **구현 기능:** `F001`, `F002` | **인증:** 로그인 필수

| 항목            | 내용                                                                                                                                                                                                                 |
| --------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 주최자 대시보드. 내가 만든 이벤트를 한눈에 파악하고 새 이벤트를 생성하는 시작점                                                                                                                                      |
| **진입 경로**   | 로그인 성공 후 자동 리디렉션, 헤더 "내 이벤트" 메뉴 클릭                                                                                                                                                             |
| **사용자 행동** | 이벤트 카드 목록 확인, 특정 이벤트 클릭으로 상세 이동, "이벤트 만들기" 버튼으로 생성 이동                                                                                                                            |
| **주요 기능**   | - 내가 만든 이벤트 목록 카드 표시 (제목, 날짜, 참여자 수, 상태)<br>- 이벤트 카드 클릭 → 이벤트 상세 페이지 이동<br>- **"이벤트 만들기"** 버튼 → 이벤트 생성 페이지 이동<br>- 이벤트 없을 때 빈 상태 안내 메시지 표시 |
| **다음 이동**   | 이벤트 카드 클릭 → 이벤트 상세 페이지, "이벤트 만들기" → 이벤트 생성 페이지                                                                                                                                          |

### 이벤트 생성 페이지

> **구현 기능:** `F001` | **인증:** 로그인 필수

| 항목            | 내용                                                                                                                                                                                                                                                    |
| --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **역할**        | 새 이벤트를 등록하는 폼 페이지. 주최자가 모임 정보를 입력해 이벤트를 개설                                                                                                                                                                               |
| **진입 경로**   | 이벤트 목록 페이지의 "이벤트 만들기" 버튼 클릭                                                                                                                                                                                                          |
| **사용자 행동** | 제목·날짜·장소 등 필수 정보 입력 후 "생성하기" 버튼 클릭                                                                                                                                                                                                |
| **주요 기능**   | - 필수 입력: 제목, 날짜/시간, 장소<br>- 선택 입력: 설명, 참여 마감일, 최대 인원<br>- 입력값 유효성 검사 (날짜 형식, 최대 인원 양수 등)<br>- **"생성하기"** 버튼 → 이벤트 저장 및 share_token 자동 생성<br>- **"취소"** 버튼 → 이벤트 목록 페이지로 복귀 |
| **다음 이동**   | 생성 성공 → 이벤트 상세 페이지, 실패 → 에러 메시지 인라인 표시                                                                                                                                                                                          |

| **다음 이동**   | 이벤트 삭제 완료 → 이벤트 목록 페이지, 수정 저장 → 현재 페이지 갱신                                                                                                                                                                                                                                             |
